# BookReaders Portal

This project is a front-end portal for exploring, selling, and requesting books.

## Pages Included
- `index.html`: Home page with quotes, authors, and book images.
- `firebaseConfig.js`: Firebase initialization script.

## Instructions
1. Host these files using GitHub Pages or Firebase Hosting.
2. Replace Firebase config if using your own project.
3. Ensure `firebaseConfig.js` is placed in the same directory as your `index.html`.
